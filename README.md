# Rishit's Portfolio 🌐

A personal portfolio website designed to showcase my skills, projects, and contact details. Built with responsive design principles and smooth animations.

## 🚀 Features
- Responsive layout optimized for all devices
- Project previews with hover effects
- Typing animation for dynamic headings
- EmailJS integration for contact form
- Deployed using Netlify

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- EmailJS
- GitHub for version control
- Netlify for deployment
